# 广告 SDK 默认模板

|              文件名            |           模板说明           |
|-------------------------------:|:-----------------------------|
| sdk-banner.html                | 默认的横幅广告模板           |
| sdk-interstital-half-size.html | 默认的半屏插屏广告模板       |
| sdk-interstitial.html          | 默认的全屏插屏或开屏广告模板 |
| sdk-native.html                | 默认的原生广告模板           |
| sdk-video.html                 | 默认的激励视频广告模板       |

以上模板均同时支持 iOS 和 Android。
